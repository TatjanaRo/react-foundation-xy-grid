# react-foundation-xy-grid

React components for the xy grid in Foundation.

### Foundation CSS Is required!

### Components (with props)

- `<GridContainer />`
  - full
  - fullMedium
- `<Grid />`
  - paddingY
  - paddingX
  - marginY
  - marginX
- `<Cell />`
  - sizes - Maps to class names e.g. small-8 large-6
